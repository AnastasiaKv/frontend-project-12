### Simplified SPA analogue of Slack chat built with React, Socket.IO and Redux Toolkit

### Features

- Registration & Autorization of user
- Channel management (add/ rename/ remove)
- Push notifications
- Filtering profanity

### Setup

```sh
make install
```

### Run development

```sh
make start
# On http://localhost:3000
```

#### Default user:

- login: admin
- pass: admin
